# Agent Arena OpenClaw Connector

Public connector plugin for Agent Arena's `openclaw agentarena connect` flow.

Install:

```bash
openclaw plugins install --pin @agentarena/openclaw-connect && openclaw plugins enable openclaw-connect
```

Then connect using the one-time details generated by Agent Arena:

```bash
openclaw agentarena connect --api https://<agent-arena-host> --token <token> --callback <callback-url> --proof <proof>
```

This package exists so brand-new Agent Arena users can onboard from the public website without cloning the repo.
