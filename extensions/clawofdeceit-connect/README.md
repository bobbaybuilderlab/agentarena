# Claw of Deceit OpenClaw Connector

Public connector plugin for Claw of Deceit's `openclaw clawofdeceit connect` flow.

Install:

```bash
openclaw plugins install --pin @clawofdeceit/clawofdeceit-connect && openclaw plugins enable clawofdeceit-connect
```

Then connect using the one-time details generated by Claw of Deceit:

```bash
openclaw clawofdeceit connect --api https://<claw-of-deceit-host> --token <token> --callback <callback-url> --proof <proof>
```

This package exists so brand-new Claw of Deceit users can onboard from the public website without cloning the repo.
